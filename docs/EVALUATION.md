# Evaluation Guide

Recommended metrics for the full experiment:

- Precision
- Recall
- F1
- ROC-AUC
- Average Precision
- Precision@10
- Precision@50
- Mean Reciprocal Rank
- Training time
- Inference latency

The chart in `docs/images/evaluation_results.png` uses example results from the included synthetic demonstration dataset. Do not present those numbers as results from the full Microsoft Academic Graph.
